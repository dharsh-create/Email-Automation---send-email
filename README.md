# Ex-07: Send Email Using Gmail

# AIM:

To send an email from a Gmail account using UiPath, including subject, body, and attachments if needed.

# INPUT:

- Gmail account email address (e.g., example@gmail.com)

- Gmail account password or App Password (for secure access)

- Recipient email address (e.g., recipient@example.com)

- Email subject (e.g., "Test Email from UiPath")

- Email body (e.g., "This is an automated email sent from UiPath")


# PROCEDURE (UiPath Steps):

1. Install UiPath.Mail.Activities →

2. Go to Manage Packages → Official → UiPath.Mail.Activities → Install.

3. Use Send SMTP Mail Message activity
